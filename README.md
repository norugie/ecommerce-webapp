Sample E-commerce web app to practice full-stack React functionalities. The project is multi-page. The home page displays a list of products that users can search through and add to cart from. Users can also click on the product name to display a modal that has more information about the product. The cart page allows users to manipulate the items that has been added to the cart. The `localStorage` mechanism has been used to keep track of the items that has been added to the cart, making sure they're available for later use. Check out the [live version](https://ecommerce.norugie.me/).
